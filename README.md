# 📦 Sistema de Controle de Estoque com Streamlit

Este é um sistema de controle de estoque desenvolvido em Python utilizando a biblioteca Streamlit. A aplicação foi criada com foco em simplicidade, praticidade e uma interface amigável para o usuário.

## 🚀 Funcionalidades

- 📋 Cadastro de produtos  
- ➕ Registro de entradas no estoque  
- ➖ Controle de saídas  
- 📊 Visualização de dados de forma interativa  
- 🔍 Consulta rápida de itens cadastrados  

## 🛠️ Tecnologias utilizadas

- Python  
- Streamlit  
- Pandas (opcional)
